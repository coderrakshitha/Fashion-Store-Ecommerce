# Fashion Store Ecommerce Web Application

## Project Overview
Fashion Store is a Java-based e-commerce web application that allows users to browse fashion products, manage shopping carts, place orders, and manage their accounts through a user-friendly interface.

## Technologies Used
- Java
- Maven
- JDBC
- Servlets
- MySQL
- HTML
- CSS
- JavaScript

## Features
- User Registration and Login
- Product Browsing
- Shopping Cart Management
- Order Placement
- User Account Management
- Database Integration
- Responsive User Interface

## Project Structure
- src/main/java - Java source code
- src/main/webapp - Frontend resources
- pom.xml - Maven dependencies and build configuration

## Future Enhancements
- Online Payment Gateway Integration
- Product Reviews and Ratings
- Order Tracking System
- Admin Dashboard
